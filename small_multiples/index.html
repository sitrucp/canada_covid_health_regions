Canada Days Since Last New COVID-19 Case By Provincial Health Regions

Interactive set of charts showing small 7-day moving average case count by provincial health regions. Color indicates days since last new case. The COVID-91 data comes from COVID-19 Canada Open Data Working Group and is updated daily.
