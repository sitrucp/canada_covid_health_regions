Canada COVID-19 Cases Small Multiple Charts By Provincial Health Regions

Interactive sort & filter set of small multiple charts of 7-day moving average COVID-91 counts by provincial health regions. The COVID-91 data comes from COVID-19 Canada Open Data Working Group and is updated daily.

  • Sort:
  • Reset
  • Region
  • Province
  • First Date (F)
  • Last Date (L)
  • Total Count (T)
  • Max Daily Count (MD)
  • # Days Since Last (DL)

Days since last case (DL): Green > 7 days | Orange 1-7 days | Red 0 days
